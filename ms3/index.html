Validation Summary

Both models were developed using LASSO-penalized logistic regression with variable selection frequency across m = 40 multiply imputed datasets. Validation included bootstrap optimism correction (B = 1000) and year-stratified leave-one-year-out cross-validation across five annual strata (2021–2025). A forward-only rolling-origin analysis confirmed consistent discrimination (size-weighted AUC 0.800). An XGBoost benchmark achieved corrected C = 0.854 but with substantially greater optimism (0.072) and poor calibration (slope 29.2).

Research tool — not for clinical decision-making. This calculator implements models from a single-surgeon, two-center study (n = 599) developed in patients already selected for radical prostatectomy. It has undergone year-stratified cross-validation and forward rolling-origin validation but not external validation in independent cohorts. Predictions should not replace clinical judgment. The model predicts any upgrading; clinically significant upgrading (GG1/GG2 to GG3+) is substantially harder to predict (corrected C = 0.653). This tool should not be used for active-surveillance eligibility decisions.
